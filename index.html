How to Install Firmament

Firmament is a Fabric-based Minecraft 1.21 mod, so installation involves setting up a mod loader before adding the mod itself. Follow these steps carefully, and if you get stuck at any point, the support channels are there to walk you through it.

  1. 1

    Install the Fabric mod loader

    Download and run the Fabric installer for the Minecraft 1.21 version Firmament targets, then create the matching Fabric profile in your launcher.

  2. 2

    Get the correct Firmament build

    Always download Firmament through a link shared officially in the Discord server, so you get a version matched to your Minecraft and Fabric setup.

  3. 3

    Place the mod file in your mods folder

    Drop the Firmament .jar file into your Minecraft mods folder alongside any required dependency mods listed with the release.

  4. 4

    Launch with the Fabric profile

    Start Minecraft using the Fabric profile you created, then join Hypixel and enter SkyBlock to confirm Firmament's features are active.

  5. 5

    Open the config to customize

    Run /firm config in-game to adjust the basics, or use /jarvis options to search through the full list of available settings and the HUD editor.

Common Issues & Fixes

Firmament doesn't load at all

Confirm you launched with the correct Fabric profile and that the Firmament .jar and its required dependencies are all in the mods folder.

Item list or recipe viewer is empty

This usually means your build is outdated or SkyBlock item data hasn't synced yet. Grab the latest release link from the Discord updates channel and reinstall.

Fishing helper isn't tracking particles while sneaking

This is a known positioning issue tied to 1.20 changes. A fix is planned but not yet released — check the Discord for status updates.

Custom texture pack items look wrong

Firmament loads models from firmskyblock:<skyblock id> first — confirm your texture pack follows that structure, or it will fall back to the vanilla model.

Game crashes on startup

Double-check your Fabric loader version matches what the current Firmament release requires, and confirm all dependency mods are present and up to date.

Not sure which fix applies to you?

Post your issue with your Minecraft/Fabric version and a screenshot in the support channel — helpers regularly diagnose issues quickly from real examples.

Tips for the Best Firmament Experience

  • Keep Firmament and Fabric both updated together — mismatched versions are the most common cause of crashes.
  • Use /jarvis options to quickly search for a specific setting instead of digging through the full config GUI.
  • Try the HUD editor early to arrange elements the way you actually play, rather than sticking with defaults.
  • If you use slot locking and cursor position saving together, test both after any update in case behavior shifts.
  • Check the feature gallery in Discord for previews of newer additions that may not be documented yet.
  • When reporting an issue, include your Minecraft version, Fabric loader version, and a screenshot — it dramatically speeds up getting help.
